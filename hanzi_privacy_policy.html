我们深知个人信息对您的重要性，将依据《中华人民共和国网络安全法》《中华人民共和国个人信息保护法》《中华人民共和国未成年人保护法》《App 违法违规收集使用个人信息行为认定方法》等法律法规，以及您所使用应用商店的相关规则，处理您在使用本应用过程中产生的相关信息。

在您开始使用本应用之前，请您仔细阅读并充分理解本政策的全部内容。当您下载、安装、启动或继续使用本应用时，即表示您已知悉并同意本政策所述各项条款。

2. 我们收集与使用的信息

2.1 本应用 不会 收集的信息

本应用为一款纯本地玩法的单机益智游戏，不需要您注册账号，也不会以任何方式向我们的服务器上传以下任何信息：

  • 姓名、性别、出生日期、身份证号、手机号、邮箱地址；
  • 设备唯一标识符（IDFA / IDFV）、MAC 地址、IMEI、设备序列号；
  • GPS 位置、IP 地址（除苹果系统层为内购、iCloud 同步等所必需的最小通信外）；
  • 通讯录、短信、通话记录、日历、照片、视频、麦克风录音、摄像头画面；
  • 第三方账号信息（微信、QQ、微博、Apple ID 的可识别身份等）。

2.2 本应用会在本地处理或保存的信息

下表所列信息全部 仅保存在您的设备本地或您 Apple ID 名下的 iCloud 中，不会被我们或任何第三方读取：

信息类型 用途 存储位置
关卡进度（已通关关卡、星级、最佳耗时） 用于在您下次启动时恢复进度，并展示在选关页 本机 Keychain；本机 Application Support 文件作为备份
金币余额 用于在游戏内"提示 / 跳过"等需消耗金币的功能扣费与展示 本机 Keychain；通过 iCloud Key-Value Store 在您同一 Apple ID 的设备间自动同步

内购订单状态 用于校验"金币包"内购是否完成并到账 Apple StoreKit 处理，仅订单状态短暂回传给本应用进行金币入账
说明以上信息均不与任何能够识别您身份的字段绑定。即便您卸载本应用，上述信息仍可能保留在 iCloud Key-Value Store 中（受 Apple 控制），以便您在重装时恢复金币余额；您可在系统设置中按章节 7 所述方式自行清除。

5. 信息的存储、共享与转让

5.1 存储地点

第 2.2 节列出的本地信息全部存储于您的设备本地或您 Apple ID 名下的 iCloud。本应用开发者不持有任何用户数据存储服务器，不存在向境外传输个人信息的情形（iCloud 由 Apple 控制，与本应用的开发者数据库无关）。

5.2 存储期限

本机数据将持续保留，直到您：

  • 主动在系统设置中清除 iCloud Key-Value Store 数据；
  • 将设备恢复出厂设置；
  • 卸载本应用（部分 Keychain 数据可能因 iOS 行为保留至下次重装）。

5.3 共享与转让

我们 不会 主动向任何第三方公司、组织或个人共享、转让您的个人信息，例外情形仅限于：

  • 事先获得您的明示同意；
  • 根据法律法规、强制性的行政或司法要求；
  • 在保护本应用、用户或公众的合法权益所必需的范围内。

5.4 公开披露

我们仅会在以下情形公开披露您的个人信息：

  • 获得您明确同意后；
  • 基于法律法规、法律程序、诉讼或政府主管部门强制性要求。

6. 信息安全保护

我们采取以下措施保护本应用涉及的信息安全：

  • 本机 Keychain 加密存储：金币余额、关卡进度等本地数据通过 iOS Keychain 服务存储，由系统提供硬件级加密保护。
  • 最小权限原则：本应用不申请任何与玩法无关的系统权限，详见第 3 节。
  • 不传输纯文本敏感信息：本应用不与任何开发者侧服务器通信，因此不存在网络传输个人信息的风险。
  • 使用系统级支付与同步：内购与 iCloud 同步均交由 Apple 系统完成，遵循 Apple 的安全规范。

尽管我们采取了上述安全措施，但请您理解，互联网环境并非绝对安全。如发生个人信息泄露事件，我们将依据法律法规要求，及时向监管部门报告并在必要时通过本应用的版本更新公告告知您。

7. 您的权利

根据《中华人民共和国个人信息保护法》，您对本应用所涉信息享有以下权利：

7.1 访问与查询

您可以在本应用的设置页 / 选关页内随时查看您的金币余额、关卡进度、设置偏好等全部信息。本应用所涉信息仅这些，并无您不可见的额外字段。

7.2 删除

您可以通过以下任一方式清除本应用涉及的信息：

  • 卸载本应用：删除本机 Application Support 中的备份文件与 UserDefaults 设置；Keychain 中的数据可能会因 iOS 机制保留至下次重装，可通过下方第二步彻底清除。
  • 清除 iCloud 数据：进入"iPhone 设置 → 您的 Apple ID → iCloud → 管理账户存储 → 汉字拆拆乐 → 删除数据"。
  • 恢复出厂设置：彻底清除设备上一切信息。

7.3 撤回同意

您可以通过停止使用本应用并按 7.2 所述方式删除已有数据，即视为撤回您对本政策的同意。撤回同意不影响在此之前已发生的处理活动。

7.4 投诉与举报

如您认为本应用的个人信息处理活动侵犯了您的合法权益，您可以通过第 10 节所列联系方式向我们投诉，或向有关监管部门进行举报。

本应用是一款全年龄段、面向汉字学习者的益智游戏，其内容适合儿童。我们高度重视未成年人个人信息的保护：

  • 本应用 不收集任何能够识别未成年人身份 的信息；
  • 本应用 不设置社交、聊天、UGC 等 涉及与陌生人交互的功能；
  • 本应用包含应用内购买（金币包），建议家长开启 iOS "屏幕使用时间 → 内容与隐私访问限制 → 购买项目"，以管控未成年人的支付行为；
